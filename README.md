## Conventions

- **Wikilinks** use Obsidian syntax. Paper links use
  `[[../FILENAME.pdf|Short alias]]` (Obsidian resolves by filename).
- Each note has YAML frontmatter with `papers:` and `related:` arrays,
  plus a body ending in a **Papers from `papers/` folder** table.
- Backlinks are implicit — every paper and every cross-topic link is
  bidirectional once Obsidian indexes the graph.

## Usage

- Open this folder as an Obsidian vault (`File → Open folder as vault`).
- Jump into [[wiki/_index|wiki/_index.md]] for the map of content.
- Graph view will reveal the densely-linked research structure.
